### 介绍
桌面计时器 
* 跨平台 
* 快捷键呼出 
* 一键开始和停止
* 不依赖鼠标

desktop-app stopwatch 
* cross-platform
* hotkey show and hide
* one key to start or stop
* mouse independent

### 使用
快捷键 hot key  
`ctrl+[`: show up  
`escape`: hide  
`enter`: start or strop
